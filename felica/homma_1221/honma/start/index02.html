カンプ作成（Photoshop）

指定のサイズの画像（著作権フリー）を選び、サイズ調整後、合成

手順

  1. 画像サイズ：600px/600px以上を6枚、600px/1240px以上を1枚選ぶ
  2. 各画像をサイズ調整：切り抜きツール
  3. 完成ファイルサイズ：2600px/1320pxを新規作成（保存名：grid.psd）
  4. 余白サイズ：40pxでグリッド設定：表示/新規ガイド
  5. 合成作業：移動ツール
  6. 背景色を設定：塗りつぶし

各要素の配置に関する基準を設定するプロパティです。
その要素が本来ある位置を基準とする方法(相対位置)や、直接配置場所を設定して配置する方法(絶対位置)を選ぶことが出来ます。

static 配置方法を指定せず、場所も指定しません。(初期値)
relative 相対位置から配置を指定します。又、基準ボックスを設定する
absolute 絶対位置から配置を指定します。
fixed 絶対位置から配置を指定し、画面をスクロールしても固定されます。

書式

基準となる親ボックスに対して

.container {
position : relative ;
}

この記述が無い場合はbodyが基準ボックスとなる

配置する要素に対して

.himawari {
position : absolute ;
top : 0 ;
left : 0 ;
}
.hybiscus{
right:0;
bottom:50px;
}

設定項目

♦ビューポート設定

画面の幅をデバイス非依存ピクセルで合わせるようページに指示が与えられる。 これにより、小さいスマートフォンと大きいパソコン用モニターのどちらでレンダリングするかを問わず、さまざまな画面サイズに合わせてページのコンテンツをリフローすることがでる。

<meta name=“viewport” content=“width=device-width,initial-scale=1”>

♦最大幅を固定値に

最大横幅をmax-widthで設定

.container{max-width:960px;}

♦横幅固定値に対して

親ボックスに対する％で指定

li{width:20%;}

♦背景画像に関して

background-sizeをcoverで設定

♦フルードイメージ（Fluid Image）

img{max-width:100%;height:auto;}

ブラウザ横幅によりCSSを切り替える

@media screen and (max-width:767px){適用するCSSを記述}
